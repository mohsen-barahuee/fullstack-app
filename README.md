## 🛠️ Getting Started

### Prerequisites

- Node.js v14+ and npm installed

### 1. Clone the Repository

```bash
git clone https://github.com/mohsen-barahuee/fullstack-app.git
cd fullstack-app

# Frontend
cd front-end
npm install

# Backend
cd ../back-end
npm install

# Backend
cd back-end
npm start

# In a new terminal
cd ../front-end
npm start
```

<p>Access the app at: http://localhost:3000</p>
