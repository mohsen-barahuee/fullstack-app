### Prerequisites

- Node.js v14+ and npm installed

### 1. Clone the Repository

```bash

cd fullstack-app

# Frontend

npm install

# Backend
cd ../back-end
npm install

# Backend
cd back-end
npm start

# In a new terminal
cd ../front-end
npm start
```
